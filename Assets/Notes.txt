Что мне надо сделать?
    При выделении объета здания или юнита или еще кого нибудь у кого есть функция ыделения, появляется интерфейс с кнопками команндами, после клика на одну из команд,
    эта команда выполняется

Какие этапы
    Выбор - отрисовка интерфейса команд - клик по команде - выполнение команды

Выбор реализован ранее. Чтобы отрисовать команды в виде кнопок нам понадобится ссами эти команды. Для этого создадим общий интерфей для всех команд.
ICommand - общий интерфейс для всех команд.
Далее создадим интерфейс для специфических команд, который будет наследоваться от общего для всех команд интерфейса.

Далее создадим интерфейс для исполнителя команд, который будет принимать в качестве параметра специфичную команду и выполнять ее.
IExecutorCommand.

Нам нужна система команд. Разобьем систему на две подсистемы. Первая часть системы это та что выполняет команды - исполнители командю Вторая часть
- отдает команды

При выборе объекта который может выполнять команды нам нужно вывести интерфейс всех команд которые есть в арсенале объекта.
Эти команды нужно отрисовать в интерфейсе пользователя. Значит в нашей системе должен быть view - интерфейса команд. В нашем случае он будет состоять
из кнопок. Кнопки будут располагаться в специальных местах для кнопок и если у объекта будет различный набор командб то будут выбираться
соответствующие кнопки.

При клике на объект который будет выполнять команды, нужно отобразить все его команды которые он будет выполнять. Для этого класс будет реализовывать
интерфейс ICommandExecute
